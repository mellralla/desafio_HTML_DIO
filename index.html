HTML: A Linguagem de Marcação para a Web

O HTML (HyperText Markup Language) é uma linguagem de marcação fundamental para a criação e estruturação de páginas da web. Desenvolvido por Tim Berners-Lee na década de 1990, o HTML permite que os desenvolvedores criem documentos que podem ser interpretados pelos navegadores e exibidos como páginas web.

Estrutura básica do HTML

Um documento HTML é composto por uma série de elementos que definem a estrutura e o conteúdo da página. Aqui estão alguns elementos essenciais:

  1. !DOCTYPE html: Esse é o primeiro elemento de um documento HTML e indica ao navegador que o documento está seguindo o padrão HTML5.
  2. html: Este elemento é o contêiner raiz de todo o documento HTML e envolve todos os outros elementos.
  3. head: O elemento contém informações sobre o documento, como título, metadados e referências a arquivos externos.
  4. body: O elemento envolve todo o conteúdo visível da página, incluindo texto, imagens, vídeos e outros elementos.

Marcação de conteúdo

O HTML oferece uma variedade de elementos para marcar e estruturar diferentes tipos de conteúdo. Aqui estão alguns exemplos comuns:

  • Títulos: Os títulos são marcados com os elementos h1 a h6, sendo h1 o mais importante e h6 o menos importante.
  • Parágrafos: Os parágrafos são marcados com o elemento p. Eles são usados para organizar blocos de texto.
  • Listas: Existem dois tipos principais de listas em HTML: listas ordenadas (ol) e listas não ordenadas (ul). Os itens da lista são marcados com o elemento li.

Elementos de mídia

O HTML também permite a incorporação de diferentes tipos de mídia em uma página. Aqui estão alguns exemplos:

  • Imagens: As imagens são incorporadas usando o elemento img. É necessário especificar o atributo src, que indica o caminho da imagem.
  • Vídeos: Os parágrafos são marcados com o elemento p. Eles são usados para organizar blocos de texto.

Links e âncoras

Os links são elementos essenciais para a navegação na web. Eles podem ser criados usando o elemento a. É necessário especificar o atributo href, que indica o destino do link.

Além disso, é possível criar âncoras em uma página usando o atributo id em qualquer elemento. Em seguida, é possível criar links que levam diretamente para essa âncora usando o atributo href com o valor # seguido pelo ID da âncora.
